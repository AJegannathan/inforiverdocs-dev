The `TEXT` function lets you change the way a number appears by applying formatting to it with format codes.

## Syntax

```javascript
TEXT(Value you want to format, "Format code you want to apply")
```

## Example

* To format the currency with a thousands separator and 2 decimals, like $6,000.84:

```javascript
TEXT(6000.839,"$#,###.00")
```

### Text Formatting Options

You can use the standard custom formatting options as shown in the table below:

| Formatting Type | Format Code    | Unformatted | Formatted      |
|-----------------|----------------|-------------|----------------|
| Number          | ##.00          | 1.598       | 1.60           |
| Large Number    | ###,###        | -1000000000 | -1,000,000,000 |
| Fraction        | ##0/##0        | 0.01        | 1/100          |
| Percent         | 0.0%           | 0.01        | 1.0%           |
| Currency        | $#,##0.00      | 10000.00    | $10,000.00     |
| Date            | yyyy,mmm,d     | 39812       | 2008,Dec,30    |
| Time            | hh:mm:ss AM/PM | 0.657384259 | 03:46:38 PM    |

You can try it yourself in the [Custom Formats Builder](https://bryanbraun.gitlab.io/custom-formats/).

Select the formatting type and enter the desired string format in the **Format Code** box. The example values change accordingly.
