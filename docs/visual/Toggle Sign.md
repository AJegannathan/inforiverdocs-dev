# Toggle Sign

The sign of the values in a single row or multiple rows can be inverted/toggled. 

Select row(s), then select **Insert  -> +/- Invert**.

![Sign of the selected rows are now inverted](/img/VisualCalculations/Sign/Sign1.png)
