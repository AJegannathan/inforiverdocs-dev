ISNAN

The ISNAN function checks for error and returns a default value if a value in the node is an error.

Syntax#

ISNAN (value1,value2)

Arguments#

value1 â€“ The node that is to be checked for error values.

value2 â€“ The value which is to replace the error values.

Return value#

value2 in the place of Error.

Example#

Consider the nodes Total Cost and No. of units to have the periods values as shown below

Total Cost10015020010030025030002253000300
No. of units1015101015251501515015

In the below example, Cost per unit is calculated by dividing Total Cost and No. of units.

To achieve this, the formula should be

ISNAN(Total Cost/No. of units,0)
